Using markdown link format in .Rmd file

From md to all other directories and formats

These links work always as they URL in paranthesis refers directly to the URL taken from the browser.

  • Markdown link from post into same directory: Text before link This is the LinkText and Text after link
  • Markdown link from post, into another post directory:Text before link This is the LinkText and Text after link
  • Markdown link from post, into tutorial directory:Text before link This is the LinkText and Text after link
  • Markdown link from post, into tutorial directory with other format:Text before link This is the LinkText and Text after link

Using shortcode link format in .Rmd file

From Rmd to all other directories and formats

  • Markdown link from dir A into same directory: Text before link This is the LinkText and Text after link
  • Markdown link from dir A into another directory B: Text before link This is the LinkText and Text after link
  • Markdown link from post into another directory: Text before link This is the LinkText and Text after link

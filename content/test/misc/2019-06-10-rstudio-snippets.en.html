Panel (as in Bootstrap 3, but shortcode)

The panel comes with different variants.

Just a Box (= default)

{{% panel %}} This is the text inside the box.
{{% /panel %}}

Panel with Title

{{% panel theme="primary" header="PRIMARY" %}} This is an info panel (blue dark) {{% /panel %}}

{{% panel theme="info" header="INFO" %}} This is an info panel (blue light) {{% /panel %}}

{{% panel theme="success" header="SUCCESS" %}} This is an success panel (green) {{% /panel %}}

{{% panel theme="warning" header="WARNING" %}} This is an info panel (orange) {{% /panel %}}

{{% panel theme="danger" header="DANGER" %}} This is an info panel (red) {{% /panel %}}

Footer Panel

{{% panel footer="title for footer panel = below text" %}} Here comes the footer text {{% /panel %}}
